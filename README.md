# NormDocsSearching_bot
## Чат-бот поиска по нормативным документам
### Описание
Телеграм-бот для поиска ответов на базе нормативных документов

На данный момент бот поддерживает такие документы как:
* [Федеральный закон "О персональных данных" № 152-ФЗ](http://www.consultant.ru/document/cons_doc_LAW_61801/ "КонсультантПлюс")

### Установка бота
#### 1) Копирование проекта
Необходимо скопировать проект из репозитория при помощи следующей команды
```
git clone git@github.com:sedoy-jango-2/normdocbot.git
```
Затем перейти в папку проекту из этой же консоли
```
cd normdocbot
```
#### 2) venv
Необходимо создать виртуальное окружение проекта во избежание конфликтов проекта. Для этого, необходимо перейти в папке с проектом после предыдущего шага необходимо вызвать следующую команду.
```
python3 -m venv env
```
Затем активируйте его в вашей папке с проектом
```
source env/bin/activate
```
#### 3) Установка зависимостей
Внутри вашего окружения установите зависимости через команду
```
pip install requirements.txt
```
#### 4) Настройка TOKEN
Необходимо настроить TOKEN телеграм-бота вашего проекта добавив его в переменные окружения
```
export BOT_TOKEN={your_token}
```
#### 5) Запуск
Для запуска проекта необходимо запустить файл *main.py* командой
```
python main.py
```
